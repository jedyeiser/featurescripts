# Footprint Feature (footprint/)

Feature for analyzing and creating footprint geometry from 3D curves. Identifies waist points, analyzes curvature, and generates scaled/transformed footprint curves.

## Files

| File | Purpose |
|------|---------|
| `analyzeFootprint.fs` | Main entry point for footprint analysis |
| `fpt_analyze.fs` | Core analysis algorithms (waist point detection, curve data) |
| `fpt_geometry.fs` | Geometric operations for footprint shapes |
| `fpt_math.fs` | Mathematical utilities (solvers, integration) |
| `fpt_constants.fs` | Constants and enums (FootprintScaleMode, etc.) |
| `predicates.fs` | Type predicates for footprint data |
| `integrateFootprint.fs` | Numerical integration for footprint properties |
| `scaleFootprint.fs` | Curve scaling and transformation |
| `arcFit.fs` | Arc fitting algorithms |

## Key Algorithms

- **Waist point detection**: Find inflection/minimum points on curves
- **Curvature analysis**: Compute and analyze curve curvature distribution
- **Scale modes**: ACCORDION, PROPORTIONAL, and other scaling strategies
- **Arc fitting**: Fit circular arcs to curve segments

## Dependencies

- `std/` - Core geometry and math modules
- `tools/` - BSpline utilities (being refactored from this codebase)

## Usage Pattern

```featurescript
import(path : "footprint/fpt_constants.fs", version : "");
import(path : "footprint/fpt_analyze.fs", version : "");

// Analyze curve for footprint properties
var curveData = buildCurveDataArray(bsplines);
var waistPoint = findWaistPoint(curveData);
```
